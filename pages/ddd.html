Espejismos y refracción de la luz

Simulación de la trayectoria de un rayo de luz en un medio cuyo índice de refracción cambia con la altura. El modelo permite observar la curvatura del rayo, la evolución del ángulo y comparar la altura mínima simulada con la altura mínima teórica.

1. Objetivo del fenómeno

Obtener la trayectoria continua de un rayo de luz en un medio heterogéneo donde el índice de refracción varía con la altura. En este caso se estudia el espejismo inferior: el rayo desciende, se curva progresivamente y alcanza una altura mínima antes de volver a elevarse.

2. Fórmulas físicas usadas

Índice de refracción variable

Se usa para modelar que el medio no es homogéneo. El índice depende solo de la altura, lo que representa una variación térmica vertical del aire.

$$n(y)=n_0+\alpha y$$

Ecuación cinemática del rayo

Relaciona el cambio de la altura del rayo con el ángulo local de inclinación respecto al eje horizontal.

$$\frac{dy}{dx}=\tan\theta$$

Ecuación dinámica del rayo

Describe cómo cambia el ángulo a medida que el rayo avanza en el medio con gradiente vertical de índice de refracción.

$$\frac{d\theta}{dx}=\frac{1}{n(y)}\frac{dn}{dy}$$

Invariante de Snell

Permite analizar el punto de retorno del rayo cuando este se nivela y se produce la reflexión geométrica asociada al espejismo.

$$n(y)\cos\theta=n(y_0)\cos\theta_0$$

Altura mínima teórica del rayo

Se obtiene imponiendo la condición de retorno $\theta=0$. Esta expresión sirve para comparar el resultado numérico con el valor teórico y calcular el porcentaje de error.

$$y_{min}=\frac{\left(n_0+\alpha y_0\right)\cos\theta_0-n_0}{\alpha}$$

Condición de existencia del espejismo

Esta desigualdad garantiza que exista una altura mínima físicamente accesible antes de que el rayo alcance el suelo.

$$\alpha>\frac{n_0\left(1-\cos\theta_0\right)}{y_0\cos\theta_0}$$

3. Método numérico: Runge–Kutta de cuarto orden (RK4)

El sistema se escribe en forma vectorial con el estado $\vec{u}(x)=\big(y(x),\theta(x)\big)$. En cada paso se calculan cuatro pendientes intermedias y se construye un promedio ponderado que mejora la precisión respecto a un método simple de Euler.

$$\vec{k}_1=F(x_i,\vec{u}_i)$$ $$\vec{k}_2=F\left(x_i+\frac{h}{2},\vec{u}_i+\frac{h}{2}\vec{k}_1\right)$$ $$\vec{k}_3=F\left(x_i+\frac{h}{2},\vec{u}_i+\frac{h}{2}\vec{k}_2\right)$$ $$\vec{k}_4=F\left(x_i+h,\vec{u}_i+h\vec{k}_3\right)$$ $$\vec{u}_{i+1}=\vec{u}_i+\frac{h}{6}(\vec{k}_1+2\vec{k}_2+2\vec{k}_3+\vec{k}_4)$$

7. Código matemático principal

Este fragmento resume el núcleo del código en Python: definición del sistema, integración con RK4 y obtención de la altura mínima simulada del rayo. 

function nOfY(y, n0, alpha) {
  return n0 + alpha * y;
}

function F(u, n0, alpha) {
  const [y, theta] = u;
  const dydx = Math.tan(theta);
  const dthetadx = alpha / nOfY(y, n0, alpha);
  return [dydx, dthetadx];
}

for (let i = 0; i < N; i++) {
  const k1 = F(u, n0, alpha);
  const k2 = F([u[0] + (h / 2) * k1[0], u[1] + (h / 2) * k1[1]], n0, alpha);
  const k3 = F([u[0] + (h / 2) * k2[0], u[1] + (h / 2) * k2[1]], n0, alpha);
  const k4 = F([u[0] + h * k3[0], u[1] + h * k3[1]], n0, alpha);

  u = [
    u[0] + (h / 6) * (k1[0] + 2 * k2[0] + 2 * k3[0] + k4[0]),
    u[1] + (h / 6) * (k1[1] + 2 * k2[1] + 2 * k3[1] + k4[1])
  ];
}

const yMinTeo = (((n0 + alpha * y0) * Math.cos(theta0)) - n0) / alpha;
const error = Math.abs(yMinTeo - yMinSim) / yMinTeo * 100;

8. Análisis y conclusiones

Validación del método

La altura mínima obtenida con RK4 debe ser muy cercana a la altura mínima teórica. Un error pequeño indica que el método reproduce adecuadamente la curvatura del rayo.

Interpretación física

Como el índice de refracción cambia con la altura, el rayo no sigue una línea recta. Su trayectoria se curva de manera continua hasta alcanzar un punto de retorno, fenómeno característico del espejismo inferior.
