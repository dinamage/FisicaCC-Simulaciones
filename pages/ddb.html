Campo eléctrico de una distribución continua de carga

Simulación del campo eléctrico producido por una barra cargada muy larga. El campo se calcula de dos formas: mediante una aproximación numérica por elementos de carga y mediante el modelo teórico de una línea infinita de carga.

1. Objetivo del fenómeno

Reproducir cómo cambia el campo eléctrico $E$ cuando se modifica la distancia $x$ entre el punto de observación y una barra con carga distribuida uniformemente. El objetivo principal es comparar el cálculo numérico con la expresión teórica y verificar que ambas curvas presenten el mismo comportamiento físico.

2. Fórmulas físicas usadas

Campo de una carga puntual

Se usa como punto de partida porque la barra se divide en muchos elementos pequeños de carga, y cada uno se trata como una carga puntual.

$$dE=\frac{k\,dq}{r^2}$$

Densidad lineal de carga

Relaciona la carga diferencial con un pequeño tramo de la barra. Permite convertir la distribución continua en elementos discretos.

$$dq=\lambda\,dy$$

Distancia al punto de observación

El elemento de carga está en la posición $y_i$ y el punto de observación está a una distancia horizontal $x$ de la barra.

$$r_i=\sqrt{x^2+y_i^2}$$

Componente horizontal del campo

Por simetría, las componentes verticales se cancelan. Por eso se suma solo la componente horizontal $E_x$.

$$E_x=\sum 2\,\frac{k\lambda x\Delta y}{(y_i^2+x^2)^{3/2}}$$

Modelo teórico de una línea infinita de carga

Sirve como referencia para validar el resultado numérico. Muestra que el campo eléctrico disminuye de forma inversamente proporcional a la distancia $x$.

$$E_{teo}=\frac{\lambda}{2\pi\varepsilon_0 x}$$

3. Método numérico: discretización de la barra

La barra se aproxima como una colección de pequeños elementos de carga separados por $\Delta y$. Para cada distancia $x$, el algoritmo suma el aporte de todos los elementos desde $y=0$ hasta un valor suficientemente grande y multiplica por 2 para considerar la simetría de la barra respecto al eje horizontal.

$$E_{num}(x)=\sum_{i=0}^{N} 2\,\frac{k\lambda x\Delta y}{(y_i^2+x^2)^{3/2}}$$ $$y_i=i\Delta y$$

En el código corregido se usa $\Delta y=0.01\,m$ y un límite $y_{max}=1000\,m$ para aproximar la barra infinita.

7. Código matemático principal

Este fragmento resume el núcleo del algoritmo: se define el modelo teórico, luego se calcula la suma numérica de los elementos de carga y finalmente se generan los datos para las tres gráficas. 

const k = 9.00e9;
const epsilon0 = 8.85e-12;
const dy = 0.01;
const yMax = 1000;

function campoTeorico(x, lambda) {
  return lambda / (2 * Math.PI * epsilon0 * x);
}

function campoNumerico(x, lambda) {
  let E = 0;

  for (let y = 0; y < yMax; y += dy) {
    E += 2 * (k * lambda * x * dy) /
      ((y ** 2 + x ** 2) ** 1.5);
  }

  return E;
}

8. Análisis y conclusiones

Comparación de curvas

El método numérico reproduce el mismo comportamiento que el modelo teórico. Las curvas disminuyen conforme aumenta la distancia $x$ y se mantienen prácticamente superpuestas en la comparación.

Interpretación física

El campo eléctrico es mayor cerca de la barra porque la distancia al punto de observación es menor. Al alejarse, el campo disminuye de forma inversamente proporcional a $x$.
