Fuerza de un fluido sobre un dique

Simulación de la fuerza total que ejerce un fluido en reposo sobre una pared vertical. La presión cambia con la profundidad, por eso se integra la fuerza diferencial a lo largo de la altura del fluido.

1. Objetivo del fenómeno

Determinar la fuerza total que un fluido ejerce sobre un dique. La presión no es uniforme: mientras más profundo está el punto analizado, mayor presión recibe la pared. Por eso no basta usar una sola presión multiplicada por el área; se debe sumar el aporte de muchas franjas horizontales del dique.

2. Fórmulas físicas usadas

Presión hidrostática variable

Se usa porque la presión depende de la profundidad medida desde la superficie libre.

$$P(y)=\rho(y)\,g\,(H-y)$$

Densidad del fluido

El enunciado considera una densidad no constante, dependiente de la altura local.

$$\rho(y)=80\,(0.250y^3-y+10.0)$$

Fuerza diferencial

Cada franja horizontal del dique aporta una pequeña fuerza al total.

$$dF=P(y)\,dA=P(y)\,L\,dy$$

Fuerza total

La fuerza total se obtiene integrando todas las franjas desde la base hasta la altura H.

$$F_T=80gL\int_0^H(0.250y^3-y+10.0)(H-y)\,dy$$

Ecuación teórica de validación

Sirve para comparar la aproximación numérica y calcular el porcentaje de error.

$$F_{teo}=80gL\left(0.0125H^3-\frac{H}{6}+5\right)H^2$$

3. Método numérico: regla del trapecio

La integral se aproxima dividiendo el intervalo $[0,H]$ en $n$ partes iguales. Los extremos se cuentan una vez y los puntos interiores se cuentan dos veces, porque cada punto interior pertenece a dos trapecios vecinos.

$$\int_a^b f(y)\,dy \approx \frac{h}{2}\left[f(y_0)+2f(y_1)+2f(y_2)+...+2f(y_{n-1})+f(y_n)\right]$$ $$h=\frac{H-0}{n}$$

7. Código matemático principal

Este fragmento muestra el núcleo del modelo: primero se define el integrando físico, luego se aplica la regla del trapecio y finalmente se compara con la ecuación teórica. 

const g = 9.81;

function funcionIntegral(y, H) {
  return (0.250 * y ** 3 - y + 10.0) * (H - y);
}

function fuerzaTrapecio(H, L, n) {
  const h = H / n;
  let suma = funcionIntegral(0, H) + funcionIntegral(H, H);

  for (let i = 1; i < n; i++) {
    const y = i * h;
    suma += 2 * funcionIntegral(y, H);
  }

  const integral = (h / 2) * suma;
  return 80.0 * g * L * integral;
}

function fuerzaTeorica(H, L) {
  return 80.0 * g * L * (0.0125 * H ** 3 - H / 6 + 5) * H ** 2;
}

8. Análisis y conclusiones

Resultado principal

Con los valores del reporte, la fuerza calculada por trapecios es muy cercana a la fuerza teórica. El error es menor al 2%, por lo que la cantidad de trapecios usada es válida para esta simulación.

Interpretación física

La fuerza aumenta al crecer H porque se incrementa la profundidad del fluido y, por lo tanto, también aumenta la presión promedio ejercida sobre el dique.
